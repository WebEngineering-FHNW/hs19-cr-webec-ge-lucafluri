Image Searcher

webeC Semester Project by Luca Fluri (Fall 2019)

Welcome to my Project the "Image Searcher".
The goal of this project was to use multiple API's of image websites,
collect returned images and display them in a grid. I added a Role Based Security, so that a user can be created and logged in
to save his or her favorite images.

The User Flow is as follows:

  • To start you can either create a new user or use the default user (username: user, password: user).
  • You can use the website without a login, but favoriting an images, requires to be logged in.
  • You can enter a searchterm and select from 3 sources (GIPHY, Unsplash, Pixabay).
  • The images are then being fetched and awaited. After all selected sources are successfully fetched
    they are displayed in a masonry grid via javascript.
  • The user can then favorite his most liked images for later if he wishes to do so.
  • The favorites can be accessed via the favorite button.
